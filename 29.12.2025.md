# Szyfr Cezara

## Wprowadzenie

Szyfr Cezara to metoda szyfrowania stworzona przez Juliusza Cezara. Działa w następujący sposób:
1. Wybierany jest klucz - liczba w zakresie 1-26,
2. Każda litera szyfrowanego tekstu jest "przesuwana" w prawo o klucz miejsc. Tj. np. przy kluczu = 3, litera A staje się literą C, litera B literą D etc. W przypadku wyjścia za alfabet wraca się na jego początek, np. Z staje się B
3. Odbiorca wiadomości otrzymuje klucz i zaszyfrowany tekst,
4. Odszyfrowanie wiadomości działa prawie identycznie, z tym że litery "przesuwane" są do tyłu o klucz miejsc.

Odszyfrowanie tekstu zaszyfrowanego tym szyfrem działa prawie identycznie, z tym że litery są przesuwane w lewo, np. przy kluczu = 3 C staje się A. 


## Implementacja

Zdefiniujmy funkcje **encrypt**, przyjmuje ona klucz o typie `int` (parametr **key**) i tekst do zaszyfrowania o typie `string` (parametr **plain_text**). Zwraca ona zaszyfrowany tekst o typie `string`.\
Zdefiniujmy również funkcję **decrypt**, przyjmuje ona klucz o typie `int` (parametr **key**) i zaszyfrowany tekst o typie `string` (parametr **encrypted_text**). Zwraca ona odszyfrowany tekst o typie `string`.\
Gdzie typy `int` i `string` dla implementacji pseudokodowej to typ liczbowy i typ tekstowy.

### Opis algorytmu

#### **encrypt**

1. Zdefiniujmy zmienną **encrypted_text** o typie tekstowym,
2. Sprawdzamy, czy **key** jest mniejsze od 1, jeżeli **key** jest mniejsze od 1 i wynosi 0, to zwracamy wartość zmiennej **plain_text**, jeżeli **key** jest mniejsze od 1 i nie wynosi 0, to przypisujemy **key** jako odwrotność aktualnej wartości **key**,
3. Sprawdzamy, czy **key** jest większe od 26, jeżeli jest, to przypisujemy **key** jako reszta z dzielenia aktualnej wartości **key** przez 26,
4. Dla każdego znaku w **plain_text** dodajemy do **encrypted_text** odpowiadający mu znak przesunięty o **key** w prawo. Jeżeli w ramach tej operacji wyszlibyśmy poza alfabet, to przechodzimy na początek alfabetu i przesuwamy `A` w prawo o liczbę pozostałą po przesunięciu za alfabet.
5. Zwracamy wartość zmiennej **encrypted_text**.

#### **decrypt**

1. Zdefiniujmy zmienną **plain_text** o typie tekstowym,
2. Sprawdzamy, czy **key** jest mniejsze od 1, jeżeli **key** jest mniejsze od 1 i wynosi 0, to zwracamy wartość zmiennej **encrypted_text**, jeżeli **key** jest mniejsze od 1 i nie wynosi 0, to przypisujemy **key** jako odwrotność aktualnej wartości **key**,
3. Sprawdzamy, czy **key** jest większe od 26, jeżeli jest, to przypisujemy **key** jako reszta z dzielenia aktualnej wartości **key** przez 26,
4. Dla każdego znaku w **encrypted_text** dodajemy do **plain_text** odpowiadający mu znak przesunięty w lewo o **key**. Jeżeli w ramach tej operacji wyszlibyśmy poza alfabet, to przechodzimy na początek alfabetu i przesuwamy w lewo `Z` o liczbę pozostałą po przesunięciu za alfabet.
5. Zwracamy wartość zmiennej **plain_text**.

### Pseudokod

        jeżeli key == 0:

### Python

```python
def encrypt(plain_text: str, key: int) -> str:
    encrypted_text = ""
    if key < 1:
        if key == 0:
            return plain_text
        key = -key
    if key > 26:
        key = key % 26
    for char in plain_text:
        shifted_letter = ""
        if char.isupper():
            shifted_letter = chr((ord(char) + key - 65) % 26 + 65)
        elif char.islower():
            shifted_letter = chr((ord(char) + key - 97) % 26 + 97)
        
        encrypted_text += shifted_letter
        
    return encrypted_text

def decrypt(encrypted_text: str, key: int) -> str:
    plain_text = ""
    if key < 1:
        if key == 0:
            return encrypted_text
        key = -key
    if key > 26:
        key = key % 26
    for char in encrypted_text:
        shifted_letter = ""
        if char.isupper():
            shifted_letter = chr((ord(char) - key - 65 + 26) % 26 + 65)
        elif char.islower():
            shifted_letter = chr((ord(char) - key - 97 + 26) % 26 + 97)
        
        plain_text += shifted_letter
        
    return plain_text
```
